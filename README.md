## How this template is intended to be used

1. Clone or fork this repository  
2. Keep folder names and document structure unchanged  
3. Populate each section with your own:
   - policies
   - risk assessments
   - model evidence
   - explainability outputs
   - fairness results
4. Export a compiled evidence pack for internal review or audit

This structure is deliberately **tool-agnostic** and **sector-neutral**.
